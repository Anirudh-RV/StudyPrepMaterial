3. Django 
Reference: 
i. Tutorial: https://www.youtube.com/watch?v=F5mRW0jo-U4&ab_channel=freeCodeCamp.org
ii. Setting up the virtual environment for Python: https://www.codingforentrepreneurs.com/blog/install-django-on-mac-or-linux
Steps:
    1. Ensure python shows python3 in your system
    2. python -m pip install pip --upgrade
    3. python -m pip install pipenv
    4. Verify: pipenv
    5. Create a new directory for the virtual environment
    6. mkdir newenv 
    7. Go into the new directory: cd newenv
    8. Set up the virtual environment: pipenv install --python 3.<version_number>
    9. To activate: pipenv shell
    10. To deactivate: deactivate
